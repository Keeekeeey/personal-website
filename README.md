# personal-portfolio

My personal portfolio site showcasing my projects, skills, and experience. Built with Next.js, React, Tailwind and Typescript and intended to emulate older window desktops on desktop and a web browser on mobile.

## Features
- **Draggable Windows** - Fully interactive windows that can be moved around the screen
- **Desktop Icons** - Click to open different sections (About, Projects, Contact, etc.)
- **Fast & Modern** - Built with Next.js for optimal performance
  
## Deployment
Deployed via Github Actions, current build is based off Main and updates automatically.

## To run locally
```
git clone https://github.com/yourname/personal-site
cd personal-site
npm install
npm run dev
```
